# antlr

[![Build](https://github.com/gsmlg-ci/antlr/actions/workflows/build.yml/badge.svg)](https://github.com/gsmlg-ci/antlr/actions/workflows/build.yml)

Docker image published to:
- `docker.io/gsmlg/antlr`
- `ghcr.io/gsmlg-dev/antlr`

## Usage

```bash
# From Docker Hub
docker pull gsmlg/antlr:latest

# From GitHub Container Registry
docker pull ghcr.io/gsmlg-dev/antlr:latest
```

## Build

```bash
docker build -t gsmlg/antlr .
```
